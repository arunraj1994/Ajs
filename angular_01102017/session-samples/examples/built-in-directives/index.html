  • $watch - Looks for a change in scope or service variables/method.
    The change happens in variable level.
  • $digest - Update the change of a scope variable when the variable is updated throught plain javascript/third party code ( non-angular code).
    The change happens in controller level.
    The $digest will invoke $watch for all the variables invoked to the same
  • $apply - Update the change of a scope variable when the variable is updated throught plain javascript/third party code ( non-angular code).
    The change happens in module level.
    The $apply will invoke $digest and that will invoke $watch for all the variables invoked to the same
